Для запуску додатку треба виконати пусковий файл TZ.exe.
Елементи інтерфейсу:
Зверху зправа є три кнопки для роботи з файлом бази данних. Його можна обновити, створити та відкрити.
Зверху зліва можна побачити шлях за яким перебуває обраний файл.
Вцентрі є дві таблиці словники та їх елементи відповідно. При виборі словника відобразяться його елементи.
Ці таблиці можна відредагувати завдяки кнопкам нижче. До кожної можна додати екземпляр, видалити чи створити новий. Кожна операція буде походити у відповідному діалоговому вікні.
Над таблицею словників є комбобокс де можна відсортувати словники.
При видаленні словника відкриється додаткова форма підтвердження, де будуть вказані всі вкладені словники, що будуть видалені.
